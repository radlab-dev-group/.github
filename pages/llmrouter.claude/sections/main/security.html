Bezpieczeństwo w centrum architektury

LLM-Router został zaprojektowany z myślą o środowiskach gdzie bezpieczeństwo danych jest priorytetem. Każdy request może być automatycznie anonimizowany przed przekazaniem do zewnętrznego dostawcy LLM.

  • Automatyczna detekcja i maskowanie danych osobowych (RODO/GDPR)
  • Wsparcie dla polskich identyfikatorów: PESEL, NIP, KRS, REGON
  • Maskowanie danych finansowych (numery kont, kwoty)
  • Usuwanie metadanych i PII z requestów
  • Walidacja checksumów dla zidentyfikowanych numerów
  • Extensible rule engine z możliwością własnych reguł
  • Force anonymization mode dla całego payloadu
  • GenAI-powered anonymization endpoint (opcjonalny)
# Przykład: Automatyczna anonimizacja export LLM_ROUTER_FORCE_ANONYMISATION=1 # Przed wysłaniem do LLM: "Jan Kowalski, PESEL: 92032145678, tel: +48 123 456 789, email: jan.kowalski@example.com, NIP: 1234563218" # Po anonimizacji: "{{MASKED}} {{MASKED}}, PESEL: {{PESEL}}, tel: {{PHONE}}, email: {{EMAIL}}, NIP: {{NIP}}"
