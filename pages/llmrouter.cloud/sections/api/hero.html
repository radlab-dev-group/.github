llm-router-api

Lightweight, extensible REST proxy for Large Language Model backends

llm-router-api provides a unified API layer that abstracts multiple LLM providers (OpenAI, Ollama, vLLM, LM Studio, llama.cpp, Anthropic) with built-in load balancing, health checks, streaming responses, connection pooling, Guardrails + Masking pipelines and Prometheus metrics.

🔌 Plugins 🛡️ Guardrails 🔒 Masking 📊 Prometheus
