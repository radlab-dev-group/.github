Configuration

Environment variables for both applications

Config Manager Variables

  • LLM_ROUTER_WEB_CFG_HOST – Bind address (default: 0.0.0.0)
  • LLM_ROUTER_WEB_CFG_PORT – Port number (default: 8081)
  • LLM_ROUTER_WEB_CFG_DEBUG – Flask debug mode (default: true)

Anonymizer Variables

  • LLM_ROUTER_WEB_ANO_HOST – Bind address (default: 0.0.0.0)
  • LLM_ROUTER_WEB_ANO_PORT – Port number (default: 8082)
  • LLM_ROUTER_WEB_ANO_DEBUG – Flask debug mode (default: true)
  • LLM_ROUTER_HOST – LLM Router API URL (e.g., http://192.168.100.65:8080)
  • LLM_ROUTER_GENAI_MODEL_ANONYMISATION – Model name for GenAI anonymization (e.g., gpt-oss:120b) 
# Example: Production deployment with custom settings
export LLM_ROUTER_WEB_CFG_HOST=0.0.0.0
export LLM_ROUTER_WEB_CFG_PORT=8081
export LLM_ROUTER_WEB_CFG_DEBUG=false

export LLM_ROUTER_WEB_ANO_HOST=0.0.0.0
export LLM_ROUTER_WEB_ANO_PORT=8082
export LLM_ROUTER_WEB_ANO_DEBUG=false
export LLM_ROUTER_HOST=http://llm-router.internal:8080
export LLM_ROUTER_GENAI_MODEL_ANONYMISATION=gpt-4

# Start with Gunicorn
./run-configs-manager.sh
./run-anonymizer.sh
