Module Overview

Two complementary web applications for router management

Key Features

  • Dual Application Architecture – Separate apps for config management (port 8081) and anonymization (port 8082)
  • Multi-User Support – Authentication system with admin/user roles and per-user project isolation
  • Configuration Versioning – Snapshot-based version control with restore capability
  • Visual Config Editor – Web UI for managing model providers, API hosts, weights, and active models
  • Text Anonymization – Three anonymization algorithms (FastMasker, GenAI, PrivMasker)
  • Interactive Chat – Real-time chat interface with model selection and anonymization toggle
  • SQLite Backend – Lightweight database with SQLAlchemy ORM for easy deployment
  • Import/Export – JSON-based configuration import/export for portability
# Start Config Manager (port 8081)
./run-configs-manager.sh

# Start Anonymizer (port 8082)
./run-anonymizer.sh

# Or run directly with Python
python app_cfg_manager.py
python app_anonymizer.py
