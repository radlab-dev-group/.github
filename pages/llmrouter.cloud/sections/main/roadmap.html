Project Roadmap

From local development to distributed blockchain-verified infrastructure. View detailed roadmap.

Level 1: Dev / Test

Foundation

Current focus: Core integrations and stabilization.

  • Integrations: LM Studio & Anthropic native support.
  • Streaming: Guardrails for streamed responses.
  • Features: RAG API (embeddings) & Prompt Injection detection.
  • UI: Enhanced ConfigsManager & Anonymizer chat.
Level 2: Production

Ecosystem

Distributed system ready for high-load environments.

  • Kafka: Async task queuing & communication.
  • Scaling: Auto-scaling logic for specific loads.
  • Agents: Integration with local agentic systems.
Level 3: Enterprise

Security

Solving critical business governance challenges.

  • SIEM: Security Information & Event Management.
  • Node Layer: Subnet routing aggregator.
  • Secure Docs: Auditable document transfer in chat.
Level 4: Research

Decentralized

Ultra-low latency and immutable trust protocols.

  • gRPC Protocol: Trusted subnet task broadcasting.
  • Blockchain: Immutable ledger for AI interactions.
  • Trust Net: The ultimate secure exchange layer.
