Comprehensive LLM Infrastructure Management

Complete platform for managing local LLM deployments (vLLM, Ollama, LM Studio, llama.cpp) and optionally cloud providers at any scale. Unified management with production-grade security and observability. Open-source architecture enables customization and community-driven enhancements. Run your models locally with full control over your infrastructure.

🔒

Built‑in Data Protection

Automatic detection and masking of sensitive data (PESEL, NIP, KRS, REGON, emails, IP addresses, etc.) followed by guard‑rail checks before forwarding requests to LLM providers. Both the masking and guard‑rail stages generate tamper‑evident, GPG‑encrypted audit logs.

⚖️

Intelligent Load Balancing

Four configurable load balancing strategies: balanced, weighted, first_available, first_available_optim (re‑use host). Optimize resource utilization and minimize latency with intelligent provider selection.

🔀

Unified Local & Cloud Routing

Single REST API supporting local providers (Ollama, vLLM, LM Studio, llama.cpp) and cloud services (OpenAI, Anthropic, Google). Automatic protocol conversion, seamless switching between local and cloud providers, and intelligent load distribution across all endpoints.

📊

Full Observability

Native Prometheus integration delivering real-time metrics: request rates, latency percentiles, error rates per provider, and health checks. Complete visibility through Grafana dashboards with automatic service discovery and alerting.

🚀

High Performance

Streaming responses with SSE support, connection pooling, Redis broker for distributed deployments. Optimized for minimal latency and maximum throughput across various deployment scenarios. Keep‑Alive system ensures models are warmed up and ready.

🔧

Easy Deployment

Infrastructure-as-code configuration via JSON and environment variables. Docker and Kubernetes ready with official images (quay.io/radlab/llm-router). Deploy in minutes with flexible configuration options for any environment.

🔌

Plugin System Architecture

Modular pipeline architecture: MaskerPipeline → GuardrailPipeline → UtilsPipeline. Each pipeline can be extended with custom plugins implementing HttpPluginInterface. Includes semantic_routing, langchain_rag, translate, and generate_questions utilities.

⚙️

Management Interfaces

Configuration Manager for centralized infrastructure control and Anonymizer interface with FastMasker and GenAI-powered text anonymization. Full versioning support with automatic snapshots.
