Security-First Architecture

LLM Router is built with data protection at its core, suitable for organizations of any size where data security matters. Every request can be automatically inspected and anonymized before transmission to any LLM provider, whether on-premise or cloud-based, ensuring compliance with GDPR, HIPAA, SOC2, and other regulatory frameworks.

  • Automatic PII detection and masking (GDPR/CCPA/HIPAA compliant)
  • Support for international identifiers: SSN, tax IDs, national IDs, business registration numbers
  • Financial data masking (account numbers, credit cards, transaction amounts)
  • Prompt injection detection and removal with security layer filtering
  • Checksum validation for detected identifiers ensuring accuracy
  • Extensible masking engine with custom rule support for industry-specific requirements
  • Enforced anonymization with configurable strategies across entire payload
  • Optional local GenAI anonymization strategy for air-gapped deployments
# ======================================= # Example: Automatic Anonymization export LLM_ROUTER_FORCE_ANONYMISATION=1 # Before transmission to LLM: "Process this data: SSN: 123-45-6789, phone: +1 555-123-4567, email: john.doe@enterprise.com, account: 4532-1234-5678-9010" # After anonymization: "Process this data: SSN: {{SSN}}, phone: {{PHONE}}, email: {{EMAIL}}, account: {{ACCOUNT}}"
