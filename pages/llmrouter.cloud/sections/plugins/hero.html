llm-router-plugins

Modular plugin ecosystem for safety, privacy, and utility

llm-router-plugins provides a unified plugin architecture with pre-built guardrails, masking rules, and utility plugins. The system is designed around a simple interface — PluginInterface with an apply(payload) method — allowing seamless extension with custom logic.
