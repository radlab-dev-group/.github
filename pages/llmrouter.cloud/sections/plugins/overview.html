Plugin Ecosystem Overview

Three categories of plugins compose the request processing pipeline: Guardrails → Masking → Utils

🛡️

Guardrails

Safety Layer

Validates content safety before processing using sequential plugins. First plugin to report is_safe=False short-circuits the pipeline.

  • Sojka-Guard (Bielik-Guard): Advanced Polish safety classifier (hate, violence, self-harm)
  • NASK Guard: NASK-PIB rule-based safety checking
  • Custom Plugins: Extend via HttpPluginInterface
🔒

Masking

Privacy Layer

Automatically detects and anonymizes sensitive PII data before transmission to LLM providers or external services.

  • FastMasker: 25+ Polish identification rules (KRS, NIP, PESEL, REGON, etc.)
  • Checksum Validation: Official algorithms minimize false positives
  • PII Classification: Context-aware NER model for unknown formats
🔧

Utils

Utility Plugins

Ready-to-use plugins for common AI workflows: semantic routing, RAG, translation, and more.

  • semantic_routing: Context-aware model selection via weighted keywords
  • langchain_rag: FAISS-based retrieval with LangChain integration
  • translate: Batch translation with configurable target languages
