Plugin Types

Three categories of plugins — each with a unified PluginInterface and apply(payload) method.

🛡️
Guardrails

Content Safety

Validates incoming payloads against safety policies. Returns (is_safe, details).

  • Sojka-Guard: Bielik-Guard model (Apache-2.0) for Polish content
  • NASK Guard: NASK-PIB rule-based classifier (CC BY-NC-SA 4.0)
  • Pipeline: Sequential — first failure rejects the request
# Pipeline config
export GUARDRAIL_STRATEGY_PIPELINE_REQUEST="sojka_guard,nask_guard"

# Returns: (False, {"label": "hate", "score": 0.71})
🔒
Maskers

PII Anonymization

Detects and replaces sensitive PII spans with placeholders like {{NIP}}, {{PESEL}}.

  • FastMasker: 25+ Polish ID formats with checksum validation
  • PII Classifier: NER model for context-aware detection
  • Validation: Official checksums minimize false positives
# Input
"NIP: 525-012-34-56"

# Output
"NIP: {{NIP}}"
🔧
Utils

Utility Plugins

Enhance payloads with semantic routing, RAG, translation, and more.

  • semantic_routing: Keyword-based model selection
  • langchain_rag: FAISS vector store for document retrieval
  • translate: Batch translation with configurable languages
# Enable utility plugins
export UTIL_PLUGINS="semantic_routing,langchain_rag"

# Semantic routing auto-selects
# optimal provider based on context
